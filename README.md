# papagaioooooos

Um jogo simples em HTML5 Canvas onde o papagaio pula em plataformas e enfrenta inimigos:

- Cobras com movimento ondulado
- Coelhos que pulam e andam
- Serras que andam em linha reta e rebatem nas paredes
- Boss final: lobo

## Como jogar
1. Abra `index.html` no navegador
2. Use as teclas `←` e `→` para se mover
3. Pressione `Espaço` para pular
4. Acumule pontos e derrote o lobo no final
